# MortgageDESK

A free UK mortgage calculator — residential & buy-to-let. Monthly payments, LTV,
stamp duty (SDLT), affordability multiples, BTL rental yield, ICR stress tests and
remortgage break-even analysis.

Fully client-side. No backend, no data collection, no accounts. Every calculation
runs in the browser.

## Run locally

```bash
npm install
npm run dev
```

Open the URL Vite prints (usually http://localhost:5173).

## Build for production

```bash
npm run build
```

Output goes to `dist/`. Preview it with `npm run preview`.

## Deploy

This is a static site — host it free on Vercel, Netlify or Cloudflare Pages.
Build command: `npm run build` · Output directory: `dist`
